kuhaku.space

Kuhaku was primarily an exercise of full stack development. Using frontend HTML/CSS/JQuery, and a NodeJS/MySQL backend, I created an unmanaged, anonymous message board that allowed users to 'pin' messages wherever they pleased. I hosted kuhaku on an Amazon EC2 instance, redirecting from a namecheap .space URL that I purchased for 88 cents (holla). Kuhaku is still up, and will continue running for the next year (until my AWS free tier expires), so feel free to visit!
